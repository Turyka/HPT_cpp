=== AAA MODEL ===
1 lépés 
username ADMIN secret Str0ngPa55w0rd
aaa new-model
aaa authentication login default local-case

aaa authentication → belépési ellenőrzésről van szó
login → amikor valaki be akar jelentkezni
default → ez az alapszabály, minden interfészre érvényes
local-case → a router saját felhasználói adatbázisát nézi meg (és a jelszó kis-nagybetű érzékeny)

- aaa local authentication attempts max-fail 10 -> Ha valaki 10-szer egymás után rossz jelszót ad meg, a fiókja zárolódik

- clear aaa local user lockout username ADMIN
